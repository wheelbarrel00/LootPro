# LootPlus
Loot+ is a lightweight, high-performance loot and combat notification overhaul...

### Recommended Addons
* **[LibSharedMedia-3.0](https://www.curseforge.com/wow/addons/libsharedmedia-3-0):** While not required, having this installed unlocks dozens of additional font and status bar textures within the LootPlus settings.
