# Loot Pro
**Loot Pro** is a high-performance, ultra-lightweight loot and combat text manager for World of Warcraft.

### Why Loot Pro?
* **Minimalist Footprint:** Optimized for low memory usage (~140KB).
* **Smart Rarity Filtering:** Configurable rarity thresholds for loot notifications.
* **Streamlined UI:** Instant tracking for Gold, Currency, XP, and Reputation.
* **Clean Mode:** Removes redundant "You receive" text for a professional look.

### Commands
* `/lp` or `/lpro` - Open the configuration menu.

### Recommended Addons
* **[LibSharedMedia-3.0](https://www.curseforge.com/wow/addons/libsharedmedia-3.0):** While not required, having this installed unlocks dozens of additional fonts from within the Loot Pro settings.
